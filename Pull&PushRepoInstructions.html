How to Push Changes to the Main Branch

1. Check the Status of Your Changes

Run the following command to check the status of your changes:

git status

This will show you the files that have been modified or added.

2. Stage Your Changes

Add the files you want to include in the commit:

git add <file_name>

To stage all changes at once:

git add .

3. Commit Your Changes

Add a meaningful commit message:

git commit -m "Your commit message here"

4. Push Your Changes to the Main Branch

Assuming your main branch is named main (verify with git branch):

git push origin main
