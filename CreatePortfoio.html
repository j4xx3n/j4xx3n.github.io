Creating a Portfolio with Github Pages

Refrences

GitHub Page you NEED this portfolio setup on your resume - Mad Hat

Process

  1. Log into github and create a public repository called username.github.io, where username is the GitHub username. If the first part of your repository does not match the username this will not work
  2. Navigate to this website and choose the hacker theme. GitHub Pages: Supported Themes
  3. Download the _config.yml, Gemfile and index.md files from the repository.
  4. Upload all three files to the portfolio repository
  5. After a few minutes the webpage template loads at https://username.github.io.
  6. I then created an html file with my first project and a directory to store images called projects.
  7. I had chatgpt turn a word document into an html file word for word and this was my result after adding some custom images from the projects directory.
