Creating a Portfolio with Github Pages

GitHub Page you NEED this portfolio setup on your resume - Mad Hat

Process

  1. Log into github and create a public repository called username.github.io, where username is the GitHub username. If the first part of your repository does not match the username this will not work ![[Pasted image 20241124071547.png]]
  2. Navigate to this website and choose the hacker theme. GitHub Pages: Supported Themes ![[Pasted image 20241124071731.png]]
  3. Download the _config.yml, Gemfile and index.md files from the repository. ![[Pasted image 20241124072345.png]]
  4. Upload all three files to the portfolio repository ![[Pasted image 20241124072201.png]]
  5. After a few minutes the webpage template loads at https://username.github.io. ![[Pasted image 20241124073031.png]]
  6. I then created an html file with my first project and a directory to store images called projects. ![[Pasted image 20241124073348.png]]
  7. I had chatgpt turn a word document into an html file word for word and this was my result after adding some custom images from the projects directory.
