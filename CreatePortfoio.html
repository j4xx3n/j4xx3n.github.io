Hash Cracking Rig Setup & Results

By: Quinn Elko

Date: 4-11-2023

Introduction

In this demonstration, I will be setting up a hash cracking rig to identify any weak or vulnerable passwords that can be easily guessed or cracked by attackers. This project demonstrates how to identify potential security risks and vulnerabilities in a system that could lead to unauthorized access, data breaches, or other security incidents. By identifying weak passwords, system administrators can take proactive measures to strengthen password policies.

For this build, I’ll be using a Dell Precision 5820 with the original case, motherboard, and CPU. I’m adding 5 Quadro M2000 GPUs, 64GB of RAM, a 500GB SSD, and a 1TB HDD. I’ll be using Ubuntu 18.04 as the operating system and Hashcat as the application to crack hashes.

Components

  • Motherboard & Case: Dell Precision 5820
  • CPU: Intel Xeon W-2223
  • GPU: (5) Quadro M2000
  • Storage: 500GB SSD & 1TB HDD
  • RAM: 64GB
  • Operating System: Ubuntu
  • Application: Hashcat

Setup

Insert the RAM, SSD, CPU, and all 5 GPUs.

Use a bootable USB to install Ubuntu 18.04.

Update and upgrade Ubuntu with the command:

sudo apt update && sudo apt upgrade -y

Install the necessary drivers for the GPUs with the command:

sudo apt install nvidia-driver-xxx

Install Hashcat with the command:

sudo apt install hashcat && sudo apt install hashcat-nvidia -y

Security

Password cracking rigs can be used for legal and illegal reasons. It is crucial that password cracking rigs only be used for ethical and legal purposes, such as pen testing with the proper authorization.

Legal & Ethical Considerations

Hash cracking should only be performed with proper authorization and for ethical and legal purposes. Unauthorized access or misuse of hash cracking can result in legal and ethical consequences. It’s important to follow all laws and regulations related to cybersecurity and to obtain permission before performing any security tests.

Recommendations

To increase the security of a system, it is recommended to enforce strong password policies, such as requiring complex passwords with a mix of letters, numbers, and symbols. It’s also important to use a password manager to ensure passwords aren’t reused across multiple accounts. Another recommendation that would greatly increase password security is to implement two-factor authentication (2FA).
